# 商品卡片样式

[toc]



## 如何使用

```html
<view class="goods-card-area" style="display: flex; flex-wrap: wrap; justify-content: flex-start;">
    <goods-card
      title="篮球"
      price='180.1'
      viewCount='12'
    />
  </view>
```

使用建议：title的字数不建议太多。price不建议太长。
